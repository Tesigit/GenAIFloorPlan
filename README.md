# 🏠 Gen AI Floor Plan - Frontend

## 📌 Project Overview
Gen AI Floor Plan is an AI-powered web application that allows users to generate intelligent floor plans through a modern, user-friendly interface. This repository contains the frontend of the application.

## 🌐 Live Demo

**Frontend:**  
https://gen-ai-floor-plan.vercel.app/

**Backend API:**  
https://genaifloorplanbackend-production.up.railway.app/

## 💻 Backend Repository

https://github.com/Tesigit/GenAIFloorPlanBackend

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript
- HTML5
- CSS3
- Axios
- Vercel

## 🚀 Features

- AI-powered floor plan generation
- Responsive user interface
- Fast API integration
- Modern React-based design

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/Tesigit/GenAIFloorPlan.git
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
